# Biblioteca LED

Biblioteca simples não bloqueante para controlar leds com ESP32 e Arduíno.

A classe "Led" permite:

• ligar LEDs continuamente;
• ligar LEDs com tempo definido;
• desligar LEDs;
• alterar estado;
• piscar continuamente;
• piscar com frequência definida;
• piscar com frequência e número de repetições definidos;
• controle não bloqueante com "millis()".
entre outros.
